What is KAR?

KAR (KLS Animation Rendering Engine) is a lightweight animation and rendering engine for Minecraft.

It uses Blockbench model groups directly and defines animations in Java without requiring GeckoLib or separate animation files.

Design Goals

  • Lightweight
  • Java-first animations
  • Blockbench group support
  • No GeckoLib dependency
  • Reusable library architecture

Versioning

KAR maintains its own version history independently of KLS Config.

Example: 

KLS Config   1.0.0
KAR Engine   1.0.0

KLS Config   1.0.0
KAR Engine   1.0.1
